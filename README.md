# Lies of P — PopTracker Pack

A [PopTracker](https://github.com/black-sliver/PopTracker/releases) pack for **Lies of P** randomizer, designed to work with [Archipelago](https://archipelago.gg/).

---

## Requirements

| Tool            | Link                                                                           |
| --------------- | ------------------------------------------------------------------------------ |
| **PopTracker**  | [Download PopTracker](https://github.com/black-sliver/PopTracker/releases)     |
| **Archipelago** | [Lies of P on Archipelago](https://archipelago.gg/games/Lies%20of%20P/info/en) |

---

## Installation

1. Download the latest release ZIP from the [Releases page](../../releases/latest)
2. Extract the ZIP into your PopTracker `packs` folder :
   - **Windows** : `%appdata%\PopTracker\packs\`
   - **Linux / macOS** : `~/.config/PopTracker/packs/`
3. Launch **PopTracker** and select the pack from the list

---

## Usage

- Open PopTracker and load the **Lies of P** pack
- Connect to your Archipelago server via the PopTracker settings
- Track your items, locations and progress in real time
